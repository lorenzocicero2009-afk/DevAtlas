# DevAtlas

> Build your own specialized local AI assistant.

DevAtlas is an open-source framework that allows anyone to build a specialized AI assistant powered by local language models and personal knowledge bases.

Instead of creating a generic chatbot, DevAtlas lets users create assistants focused on specific domains such as Linux, Python, networking, documentation, books, games, research, or any custom collection of documents.

Everything runs locally, giving users full control over their data and models.

---

## Features

- Local AI (no cloud required)
- Multiple LLM support (Gemma, Qwen, Llama...)
- Retrieval-Augmented Generation (RAG)
- Automatic document indexing
- Modular architecture
- Plugin-ready
- Offline knowledge base
- Easy customization
- Open Source

---

## Supported Documents

- TXT
- Markdown
- PDF
- HTML
- DOCX
- JSON
- CSV

More formats coming soon.

---

## Project Goals

DevAtlas is designed to be a framework rather than a single AI assistant.

Every user should be able to:

- add their own datasets
- customize prompts
- choose their preferred LLM
- enable or disable knowledge modules
- create domain-specific assistants

without modifying the core code.
