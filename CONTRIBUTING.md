# Contributing to DevAtlas

Thanks for your interest in DevAtlas!

## How to contribute

- Fork the repository
- Create a new branch
- Make your changes
- Test your code
- Open a Pull Request

## Coding style

- Use Python 3.12+
- Follow PEP8
- Write clear comments
- Keep functions small and readable

## Reporting bugs

Please open an Issue describing:

- Operating system
- Python version
- Error message
- Steps to reproduce
